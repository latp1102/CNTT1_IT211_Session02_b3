HTTP Status Code giúp client hiểu chính xác kết quả xử lý request của server.
Nếu API chỉ trả về null thay vì ResponseEntity với HttpStatus.NOT_FOUND, server vẫn trả 200 OK. Khi đó client không thể biết item không tồn tại hay dữ liệu rỗng hợp lệ, gây khó khăn cho việc xử lý lỗi.
Vai trò của Jackson Dataformat XML
Content Negotiation cho phép client chọn định dạng dữ liệu thông qua header Accept (JSON hoặc XML).
Spring Boot mặc định chỉ hỗ trợ JSON. Vì vậy cần thêm dependency Jackson Dataformat XML để Spring Boot tự động chuyển đổi object Java sang XML khi client gửi:
Accept: application/xml
Nếu không có thư viện này, API sẽ không trả được XML hoặc gây lỗi 406 Not Acceptable.
